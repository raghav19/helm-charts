
WLS-DB TLS Certificate Generator  
===============================

A Helm chart for generating tls certs for WLS database deployment


## Configuration

The following table lists the configurable parameters of the WLSDB-CERT-GENERATOR chart and their default values.

| Parameter                | Description             | Default        |
| ------------------------ | ----------------------- | -------------- |
| `nameOverride` | The name for WLSDB-CERT-GENERATOR chart (Default: .Chart.Name) | `""` |
| `securityContext.wlsdbCertGeneratorInit.fsGroup` |  | `1001` |
| `securityContext.wlsdbCertGenerator.runAsUser` |  | `1001` |
| `securityContext.wlsdbCertGenerator.runAsGroup` |  | `1001` |
| `securityContext.wlsdbCertGenerator.capabilities.drop` |  | `["all"]` |
| `securityContext.wlsdbCertGenerator.allowPrivilegeEscalation` |  | `false` |
